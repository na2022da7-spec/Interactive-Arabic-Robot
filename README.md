# Interactive-Arabic-Robot


**Nebras** is an Arabic interactive robot developed to communicate with users using speech recognition, artificial intelligence, and servo motor movements. The project combines Python and Arduino to create a simple human-robot interaction experience.

## Features

* Arabic speech recognition.
* Arabic text-to-speech responses.
* AI-powered conversations.
* Servo motor control for head and arm movements.
* Interactive gestures while speaking.

## Technologies Used

* Python
* Arduino
* Servo Motors
* SpeechRecognition
* gTTS (Google Text-to-Speech)
* Google Gemini API

## How It Works

1. The robot listens to the user's voice.
2. The speech is converted into text.
3. The AI generates a response.
4. The response is converted into Arabic speech.
5. The Arduino moves the robot's head and arms while it speaks.

## Project Goal

The goal of Nebras is to create a simple Arabic-speaking robot that demonstrates the integration of artificial intelligence, speech processing, and robotics.
